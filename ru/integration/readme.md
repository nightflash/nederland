# Интеграция

Формальная часть интеграции состоит из сдачи интеграционных экзаменов, получения ПМЖ и/или гражданства Нидерландов.

- [ПМЖ](./prp.md) — пятилетний срок, экзамены, подача заявления через My IND и получение карточки ВНЖ.
- [Гражданство](./citizenship.md) — подача заявления через муниципалитет, рассмотрение IND, церемония натурализации и получение паспорта.

ПМЖ не является обязательным этапом перед натурализацией. Однако kennismigranten и зависимым партнерам рекомендуется сначала одновременно получить ПМЖ, чтобы обезопасить свой статус и не попасть на обязательный B1-трек.
